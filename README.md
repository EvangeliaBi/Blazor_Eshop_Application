# Blazor E-Shop Application

This project is a mini e-commerce web application developed as part of the Object Oriented Programming module.

## Technologies Used
- C#
- Blazor Server
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- HTML / CSS

## Description
The application simulates an online electronics store and supports two user roles:
- **Admin**: Product and order management (CRUD operations)
- **Customer**: Product browsing, shopping cart, wishlist and order placement

The system implements authentication and authorization, secure password hashing,
real-time UI updates using Blazor Server and a PostgreSQL relational database.

## Features
- User authentication (Admin / Customer)
- Product management (CRUD)
- Shopping cart
- Wishlist
- Order history
- Role-based access control

## Database
PostgreSQL is used as the database management system.  
Entity Framework Core is responsible for database access and migrations.
